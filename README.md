# QCD-Flux-Tube
A single-scale phenomenological cavity model of the QCD flux tube: predicting a bounded chiral-limit string-breaking distance (~1.24 fm) and related confinement observables from one transverse radius R₀ ≈ 0.81 fm.
A single transverse scale for the QCD flux tube
Code and data accompanying the note 
"A single transverse scale for the QCD flux tube: a bounded chiral-limit string-breaking distance." 
The flux tube is modelled as a cylindrical cavity of transverse radius R0 ˜ 0.81 fm, with transverse 
modes quantised to Bessel zeros. From this single input the model predicts a finite chiral-limit 
string-breaking distance (1.24 fm) and reproduces the dual-gluon mass, penetration length, and gluonic-excitation 
ratios. Includes the mpmath scripts that reproduce all equations, the data table, and the figures.
